# Product Mission

## Pitch

SexyLove is an ultra-premium personal companion application defined by its **Dual-Layer Cognitive Architecture**. It provides a "Superficial" Public Facade—a high-end, glassmorphic Notes and Todo utility—that conceals a deeply encrypted "Hidden Vault" for sensitive memories, locations, and identities.

## Users

### Primary Customers

- **Privacy-Conscious Individuals**: People who want to keep personal diaries or location history safe from prying eyes via obfuscation.
- **Aesthetic Seekers**: Users who appreciate high-end design, smooth animations, and premium UX that doesn't "look" like a security tool.

### User Personas

**The Keeper** (20-45)

- **Role:** Personal User
- **Context:** Private emotional journaling and memory management.
- **Pain Points:** Traditional security apps look suspicious and technical.
- **Goals:** A beautiful app that feels like a standard utility but holds a deep secret.

## The Problem

### The "Security vs Beauty" Dilemma

Existing apps are either beautiful but insecure, or secure but aesthetically generic/suspicious (the "Calculator Vault" trope). This forces users to either sacrifice aesthetics or risk exposure.

**Our Solution:** A dynamic, multi-layered approach:

1.  **Public Facade (Chameleon)**: A fully functional notes/todo app that serves as the "Superficial" layer.
2.  **The Trigger**: A hidden gesture (3 rapid clicks on the title) that initiates the Auth Gateway.
3.  **Hidden Vault**: A deeply integrated, encrypted environment for "Geografía del Deseo" (Maps), "Fantasies", and "Ephemeral Chat".

## Differentiators

### Chameleon Persona System

Unlike static apps, SexyLove uses Native Android Activity Aliases to dynamically switch its App Icon and Name (e.g., from "Notas" to "Azulinaa" or "Muaah"), making it virtually undetectable to casual observers.

### Cinematic Experience

Powered by GSAP and React 19, the UI provides a "World-Class" feel with glassmorphism, micro-animations, and physics-based interactions that transcend standard app expectations.

## Key Features

### Core Features

- **The Hidden Vault:** AES-256 client-side encrypted storage for sensitive entries.
- **Chameleon Identity:** Native dynamic icon switching to obfuscate the app's purpose.
- **Geografía del Deseo:** Private geographical memory visualization via MapLibre GL.
- **Eternity Sync:** A real-time, conflict-resilient protocol for multi-device integrity.
- **Multi-Identity:** Support for multiple user profiles within a single vault.
- **Block-Based Editor:** A sophisticated editor (Style Toolbar, Smart Enter) for the Public Facade.
