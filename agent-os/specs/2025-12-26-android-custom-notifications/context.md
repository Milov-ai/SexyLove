# Context: Android Custom Notifications

## Goal

Verificar e implementar la posibilidad de notificaciones altamente personalizadas en Android usando Capacitor (o alternativas si es necesario) para SexyLove. Esto incluye investigar soporte para iconos personalizados, sonidos, y posibles "Custom Layouts" si Capacitor lo permite directamente o mediante plugins específicos.

## Definition of Done

- [ ] Análisis de viabilidad completado.
- [ ] Plugin de notificaciones configurado (Push o Local).
- [ ] Notificación de prueba enviada con éxito a un dispositivo/emulador Android.
- [ ] Personalización (icono/sonido) verificada.
- [ ] Todas las tareas en `tasks.md` completadas.
- [ ] El build de Android pasa correctamente.
