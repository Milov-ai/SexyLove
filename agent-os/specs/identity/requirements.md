# Spec: Chameleon Identity

## Status: Active / Core Feature

## Description

Chameleon Identity allows SexyLove to assume different "facades" (Public side of the app).

## Core Requirements

1.  **Multiple Facades**: Utility, Language, Music, Food.
2.  **Dynamic Theming**: Color palettes and navigation switching.
3.  **Identity Switch**: Remote synchronization via Supabase.

## Implementation Details

- **Manager**: `ChameleonManager.ts`.
- **Sync**: `ChameleonRemote.ts`.
